Installation Instructions

Installing on Windows

  1. Download the FiraCode font package from the link above
  2. Extract the ZIP file to a folder
  3. Right-click on each .ttf file and select "Install" or "Install for all users"
  4. Restart your code editor or IDE
  5. Enable FiraCode in your editor's font settings

Tip: In VS Code, add "editor.fontFamily": "Fira Code" and "editor.fontLigatures": true to your settings.

Installing on macOS

  1. Download the FiraCode font package
  2. Extract the downloaded archive
  3. Double-click on each .ttf file
  4. Click "Install Font" in Font Book
  5. Restart your code editor
  6. Select FiraCode in your editor's preferences

Alternative: Use Homebrew: brew tap homebrew/cask-fonts && brew install font-fira-code

Installing on Linux

Ubuntu/Debian

sudo apt install fonts-firacode

Arch Linux

sudo pacman -S ttf-fira-code

Fedora

sudo dnf install fira-code-fonts

Manual Installation

  1. Download and extract the font files
  2. Copy .ttf files to ~/.local/share/fonts/
  3. Run fc-cache -f -v

Editor Configuration

Enable ligatures in your favorite code editor

VS Code

"editor.fontFamily": "Fira Code",
"editor.fontLigatures": true

Sublime Text

"font_face": "Fira Code",
"font_options": ["gray_antialias"]

Atom

editor:
  fontFamily: "Fira Code"
  fontLigatures: true

IntelliJ IDEA

Settings → Editor → Font
Font: Fira Code
Enable ligatures: ✓
