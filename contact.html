Frequently Asked Questions

How do I install FiraCode?

Visit our Download page for detailed installation instructions for all platforms.

Is FiraCode really free?

Yes! FiraCode is completely free and open-source under the SIL Open Font License.

Which editors support FiraCode?

FiraCode works with VS Code, Sublime Text, Atom, IntelliJ IDEA, and most modern code editors.

Can I contribute to FiraCode?

Absolutely! Check out our GitHub repository to get started.
